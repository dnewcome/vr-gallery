# VR Art Gallery

A virtual reality art gallery built in Unity for Meta Quest. Walk through a spacious gallery space and experience art in an immersive 3D environment.

## Overview

This project creates an online VR art gallery — a large open gallery building where visitors can view artwork displayed on the walls and throughout the space.

## Tech Stack

- **Engine:** Unity (URP — Universal Render Pipeline)
- **Platform:** Meta Quest (OpenXR)
- **XR:** XR Interaction Toolkit 3.3.0, XR Hands 1.7.1

## Gallery

The gallery is a 100 ft × 100 ft barn-style building featuring:

- Gable roof with a ridge height of 22 ft
- 4 skylights with warm lighting
- 7 structural trusses
- 20 ft roll-up door entrance on the front

## Getting Started

### Requirements

- Unity 6 or later
- Meta Quest Link or Quest headset
- XR Interaction Toolkit package

### Setup

1. Clone the repo
2. Open the project in Unity
3. Open `Assets/Scenes/BasicScene.unity`
4. To rebuild the gallery, use the menu: **Gallery > Build VR Gallery**

## Project Structure

```
Assets/
  Scenes/        # Unity scenes
  Editor/        # Editor scripts (GalleryBuilder.cs)
```
